# Movie Recommendation System

## Introduction
This is a **Movie Recommendation System** built with Python.  
It suggests movies to users using:  
- **Content-based filtering** ‚Üí recommends similar movies by description, cast, director, or genre.

---

## üõ† Tech Stack
- **Python 3.10+**
- **Pandas, NumPy** ‚Üí data cleaning and processing
- **Jupyter Notebook** ‚Üí exploration & experiments
- **Scikit-learn** ‚Üí similarity calculation (content-based)
- **difflib** ‚Üí fuzzy matching for movie title search
- **FastAPI** ‚Üí RESTful API backend
- **Uvicorn** ‚Üí run API server
- **Railway** ‚Üí cloud deployment for hosting the app
